## Required
1.  Ruby https://www.ruby-lang.org/en/
2. Jekyll https://jekyllrb.com/
## Setup
1. Run ``$ bundle``
## Development
1. Run ``$ bundle exec jekyll serve``
## Production
1. Run ``$ jekyll build``
